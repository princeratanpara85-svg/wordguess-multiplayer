📖 How to Play

🎯 Objective

Guess the hidden word before time runs out! Compete with 2-4 friends to see who can figure out the word first.

🎮 Getting Started

  • Set your name in Profile before playing.
  • One player creates a game and gets a 6-digit code.
  • Other players use this code to Join the game.
  • The creator chooses word length (5, 6, or 7 letters) and difficulty (Easy, Medium, Hard).
  • Minimum 2 players, maximum 4 players.

🔤 Guessing Rules

  • Type a meaningful English word with the correct number of letters.
  • Only alphabets are allowed — no numbers or symbols.
  • Random sequences of letters are not accepted — only real words!
  • You can guess continuously — no need to wait for other players.

🎨 Color Coding

A

Green = Correct letter in the correct position

B

Yellow = Correct letter but in the wrong position

C

Red = Letter is not in the word at all

⏱️ Rounds & Timing

  • Each round lasts 90 seconds.
  • If someone guesses correctly, they win the round and a new round starts.
  • If nobody guesses in time, the word is revealed and a new round begins.

🔁 Duplicate Letters

If the hidden word has a letter appearing more than once (e.g., "SYSTEM" has two "S"), all players will receive a hint like: "two same alphabets 'S' are used in this word".

🏆 Winning

  • The player who guesses correctly first gets a point and sees: "Congratulations you have won!"
  • Your game history tracks how many words you've guessed correctly.
